## Configuration

### Import global

```js
// main.js
import { createApp } from 'vue'
import SurgeUpDS from '@surgeup/ds-vue'
import '@surgeup/ds-vue/style.css'
import App from './App.vue'

const app = createApp(App)
app.use(SurgeUpDS)
app.mount('#app')
```

### Import s√©lectif

```vue

import '@surgeup/ds-vue/style.css'
</script>

<template>
  <SuButton variant="primary">
    Mon bouton
  </SuButton>
</template>
```
